Bitwarden Passwordless.dev - Client-JS (local build)

This example loads the SDK from /passwordless-client-js/iife/passwordless.iife.js, which is served straight from this repo's dist/ output. Run npm run build at the repo root before starting the example so the bundle exists.

To run this example you must first add your API key in index.html and your API Secret in .env.
